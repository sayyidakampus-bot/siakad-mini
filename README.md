# SIAKAD Mini

Sistem Informasi Akademik Mini berbasis PHP Native dan MySQL.

## Fitur

### Level 1
- Login & Logout
- Session Guard
- CRUD Dosen
- Upload Foto
- Soft Delete
- Search
- Pagination
- CSRF Protection
- Prepared Statement
- MIME Validation

### Level 2
- OOP Architecture
- RBAC (Admin & Operator)
- Restore Trash
- Many-to-Many Relation
- Transaction
- Filter & Sorting
- JOIN & GROUP BY

### Level 3
- Dashboard Statistik
- Export CSV
- Audit Log
- Rate Limiting Login

---

## Struktur Folder

```bash
siakad-mini/
├── config/
├── src/
├── public/
├── uploads/
└── README.md
```

---

## Cara Menjalankan

1. Jalankan XAMPP
2. Import database `seed.sql`
3. Simpan project di folder `htdocs`
4. Akses:

```txt
http://localhost/siakad-mini/public
```
